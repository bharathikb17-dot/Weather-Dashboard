\# 🌦️ Smart Weather Dashboard



\## Features

\- 🌡️ Current Weather

\- 📅 5-Day Forecast

\- 🌫️ Air Quality Index (AQI)

\- 📊 Charts (Matplotlib)

\- 💾 Save search history (CSV)

\- 🔐 Secure API using .env



\## Setup



pip install -r requirements.txt



Create .env:

API\_KEY=your\_api\_key



Run:

streamlit run app.py
